# Быстрый старт: GigaChat + SRAF

## ✅ Все уже настроено! Начните здесь

SRAF полностью интегрирован с GigaChat API и готов к использованию.

## Одна команда для начала

```bash
cd /home/user1/lab_agent
./sraf.sh run "Напиши функцию для сортировки" --no-verify-ssl
```

## Основные команды

### Простая задача
```bash
./sraf.sh run "Напиши приветствие на русском" --no-verify-ssl
```

### Интерактивный чат
```bash
./sraf.sh chat --no-verify-ssl
# Введите задачу, :q для выхода
```

### Demo режим (без интернета)
```bash
./sraf.sh run "тест" --demo
```

## Как это работает

1. **Извлечение инструкций** - система понимает вашу задачу
2. **Выполнение** - агент выполняет инструкции с инструментами
3. **Оценка** - система оценивает результат
4. **Уточнение** - при необходимости улучшает результат

## Если что-то пошло не так

### Ошибка: "Permission denied"
```bash
chmod +x sraf.sh
```

### Ошибка про SSL
Это нормально для корпоративного прокси. Используйте флаг `--no-verify-ssl`.

### Ошибка про GIGACHAT_CREDENTIALS
```bash
# Переменные должны загруститься автоматически
# Но если не загрузились:
export $(cat .env | grep -v '#' | xargs)
```

## Параметры команды

```bash
./sraf.sh run "ЗАДАЧА" [ОПЦИИ]

Опции:
  --demo                  # Demo режим (мгновенно, нет API)
  --no-verify-ssl         # Отключить проверку SSL
  --max-steps N           # Максимум итераций (по умолчанию 5)
  --max-attempts N        # Максимум попыток (по умолчанию 3)
  --base-prompt-file FILE # Файл с пользовательским промптом
```
